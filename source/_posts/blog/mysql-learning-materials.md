## 前言

MySQL 是当前最流行的开源数据库，国内外众多公司基本都使用 MySQL 来支撑核心业务，学习 MySQL 能够帮助我们更好地完成日常工作，提升 SQL 开发和相关问题排查的能力。本文旨在收集整理 MySQL 学习的相关文档资料，希望能够通过这些资料，系统地学习掌握 MySQL 技术。
